Hello,

twoPointAPI is a java application with spring framework hosted in aws using beanstalk connected to RDS running postgreSQL. Code is written in java 18, but during creation of jar file it's converted to java 8 for beanstalk.

Code creates/access a transaction database. A single transaction has transactionID (created automatically by table, but can be specified), userID, product, and amount.

To access api and see this page follow:
Twopointapi-env-1.eba-vgvyuxmu.eu-west-1.elasticbeanstalk.com

To get all transactions:
Twopointapi-env-1.eba-vgvyuxmu.eu-west-1.elasticbeanstalk.com/api/v1/transaction/all

To post a single transaction use any crud app using following json format:

Please note transactionID is added by the database.

{
"userId" : 777,
"product" : "red whale",
"amount" : 19.99
}

url:
Twopointapi-env-1.eba-vgvyuxmu.eu-west-1.elasticbeanstalk.com/api/v1/transaction/add
